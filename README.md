# TunedUp

A simple guitar tuner app built with Next.js, React Bootstrap, and the Web Audio API.

## Features

- Live tuning detection
- Custom tuning controls
- Supabase implemented presets and sign-in

## Stack

- Next.js
- React Bootstrap
- Tailwind CSS
- Supabase
